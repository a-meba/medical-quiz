📖 프로젝트 소개

의학용어의 기초가 되는 1~3장(어근, 접두사, 접미사, 결합형 등) 100문제를 마스터하기 위해 제작되었습니다. 단순 객관식이 아닌 주관식 입력 방식을 채택하여 실제 암기력을 극대화하는 데 중점을 두었습니다.

✨ 주요 기능과 특징

3가지 학습 모드 지원

정주행 🚀: 1번부터 100번까지 순서대로 꼼꼼하게 학습

전체 랜덤 🎲: 100문제를 무작위로 섞어 실력 점검

랜덤 20제 🔥: 바쁜 시간, 핵심 20문제만 빠르게 퀴즈

스마트 채점 시스템: 공백, 대소문자, 특수문자 예외 처리를 통한 유연한 정답 체크

오답 복습 기능: 틀린 문제만 모아서 다시 풀 수 있는 리트라이 기능 제공

실시간 피드백: 문제 풀이 즉시 정답 여부와 해설 확인 가능

진행도 표시: 프로그레스 바를 통해 학습 성취도 시각화

🛠 사용되는 기술 및 언어

Language: HTML5, JavaScript (ES6+)

Styling: Tailwind CSS (Modern UI Framework)

Deployment: GitHub Pages

Font: Google Noto Sans KR
